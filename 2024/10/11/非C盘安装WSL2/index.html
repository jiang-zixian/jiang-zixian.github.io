非C盘安装WSL2

非C盘安装WSL2

第一种方法（失败，下载的WSL2不能git clone）

install WSL2

uninstall ubuntu

在电脑中直接卸载即可

install ubuntu in F:

win 终端运行：

之后设置自己的username和password即可

问题:WSL2 请启用虚拟机平台 Windows 功能并确保在 BIOS 中启用虚拟化

之后重启电脑即可

第二种方法（安装在C盘之后迁移至F盘）

下载 WSL2

终止正在运行的wsl

将需要迁移的Linux，进行导出

导出完成之后，将原有的分发进行卸载

然后将导出的文件放到需要保存的地方，进行导入

启动WSL并进入原来的user
